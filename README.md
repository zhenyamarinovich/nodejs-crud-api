## Getting started:

- clone this repo (git clone);
- switch to dev branch (git checkout dev);
- install packages (npm i).

## Application mods:
- npm run start:dev
- npm run start:prod"
- npm run start:multi
- npm run lint
- npm run format
- npm run test

## Endpoints:
- GET api/users
- GET api/users/{userId}
- POST api/users
- PUT api/users/{userId}
- DELETE api/users/{userId}
